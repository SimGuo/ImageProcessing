*extract.cpp:*

1. 使用一部分数据作为训练集，对其中每张图片提取SIFT特征。
2. 将所有提取出来的SIFT特征混在一起，用kmeans将其分为K类，也就是K个词汇。
3. 用生成的词汇集去处理原先的图片集合，对原先的每张图片重新计算出一个K维的特征向量。每个维度是词汇集中词汇出现的次数。
4. 用生成的K维特征向量，写到文件里面。
>注意：这个k维向量在标准化之后就变成了和为1的频率，所以每一维都是一个浮点数
